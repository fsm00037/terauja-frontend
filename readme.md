# Terauja Frontend - Psychologist Dashboard

This is the frontend application for psychologists in the **Terauja** project. It provides a comprehensive dashboard for monitoring patient progress, managing sessions, and analyzing clinical data.

## 🚀 Overview
The dashboard is built with **Next.js** and features a modern, responsive interface for clinical management. It communicates with the Terauja Backend to provide real-time updates and AI-driven insights.

### Key Features
- **Patient Monitoring**: Visual representation of EMA (Ecological Momentary Assessment) data.
- **Session Management**: Tools to schedule and log patient sessions.
- **Clinical Notes**: Secure storage and management of therapist notes.
- **Advanced Analytics**: Detailed charts and metrics for patient progress using Recharts.
- **Real-time Notifications**: Integration for alerts and updates.

## 🛠️ Technology Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide React](https://lucide.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Charts**: [Recharts](https://recharts.org/)
- **Forms**: React Hook Form + Zod
- **Language**: [TypeScript](https://www.typescriptlang.org/)
